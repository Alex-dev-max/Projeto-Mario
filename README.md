<h1 align="center">🚰 Landing Page - Empresa de Encanamento</h1>

<p align="center">
Projeto desenvolvido com foco em uma landing page moderna, responsiva e visualmente atrativa para uma empresa de serviços de encanamento.
</p>

---

## 📖 Sobre o Projeto

Este projeto consiste em uma página inicial criada para apresentar uma empresa de encanamento de forma profissional e moderna.

A aplicação conta com:

- 🎥 Vídeo de fundo para maior impacto visual  
- 🏷️ Logotipo personalizado  
- 📝 Texto informativo sobre os serviços  
- 📱 Layout totalmente responsivo  

O objetivo do projeto é oferecer uma experiência agradável ao usuário, destacando os serviços da empresa de maneira clara e visualmente atrativa.
